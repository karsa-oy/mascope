1. DataTab.Vue

The datatab component was used originally for visualization of data from .h5 files from local drive. The datatab allows listing all the samples in drive, selecting and processing samples, displaying 
detected targets and visualizaing them, applying different detection parameters values via sliders saving new values etc. 

 

The images below shows the different states of datatab functionality. 
a. The initial datatab layout / state

 

b. The selected file is processed after data path is set, peaklist is set, filter is applied(default), a single samples is selected and processed

 

c. The datatab layout state when filter is not applied (all target lista are displayed)

 

d. The datatab layout state when multiple samples are processed, heatmap is changed to scatter plot

 

e. The datatab feature which lets user to edit the peaklist which was selected in datatab.
