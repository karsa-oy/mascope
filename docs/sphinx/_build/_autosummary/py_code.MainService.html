py_code.MainService¶

Main Service

This script runs the main service for Karsa Tarkka TOF system.

MainService runs a FlaskSocketIO server, into which other services connect as clients. It also connects to KAcquisition and KSignalProcessor instances in InstrumentService via Pyro4.

When run, this script will configurations from .env file. Then it tries to initialize Pyro4 proxy objects for KAcquisition (and KSignalProcessor) in InstrumentService, thus InstrumentService is required to be running. Then the script enters main() function.
