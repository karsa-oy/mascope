on_created(event)¶

Action to be taken when new file is detected

It updates the attributes 'new_file' and 'last_complete', copies the 'last_complete' file into 'dl_path' if defined, puts the 'last_copied' file into 'file_queue' if defined and lastly appends 'last_copied' to 'datapool' if defined.

event (FileSystemEvent) -- Detected event in the file system
