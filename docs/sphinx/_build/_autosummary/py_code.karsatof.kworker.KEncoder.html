encode(**kwargs)¶

Function that performs SparseCoding for a segment of mass spectrum

Parameters

  • s0 (inst) -- First sample number

  • s1 (inst) -- Last sample number

  • spec (array) -- Signal to encode

Returns

3-tuple is returned with (i0, i1, code), where 'i0' and 'i1' are the rows of the dictionary and 'code' is the processing result. If subsampling=1, i0==s0 and i1==s1.
