py_code.karsatof.kcode.thresholding¶

py_code.karsatof.kcode.thresholding(y, threshold, hysteresis, fill_gaps)¶

Thresholding with hysteresis

Parameters

  • y (array) -- Signal for which to do thresholding

  • threshold (double) -- Threshold value, for negative threshold values the algorithm looks for values below abs(threshold)

  • hysteresis (double) -- If previous value is above threshold, the consecutive one need be only above (threshold-hysteresis) to still be counted

  • fill_gaps (int) -- Connect groups in the output if the gap in between is less or equal to this value

Returns

ind -- Array of boolean values indicating whether the inout signal at the index is above (or below) threshold

Return type

array
