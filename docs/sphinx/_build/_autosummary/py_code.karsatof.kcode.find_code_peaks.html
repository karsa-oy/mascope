py_code.karsatof.kcode.find_code_peaks¶

py_code.karsatof.kcode.find_code_peaks(y, x=None, min_height=1e-05)¶

Find peaks in signal using the algorithm 'find_peaks' from 'scipy.signal'.

Parameters

  • y (array) -- Signal to search for peaks

  • x (array, optional) -- Dimension of the signal. The default is None.

  • min_height (double, optional) -- Minimum peak height. The default is 1e-5.

Returns

peaks -- Found peak locations in x if given, otherwise as indices of y.

Return type

array
