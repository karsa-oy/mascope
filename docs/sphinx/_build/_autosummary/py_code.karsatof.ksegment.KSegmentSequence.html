py_code.karsatof.ksegment.KSegmentSequence¶

class py_code.karsatof.ksegment.KSegmentSequence(borders, max_seq_len=inf, gap_thresh=1, max_distance=1, min_len=3)¶

Bases: object

Class representing a sequence of "codes" for a spectral segment.

The segment sequence is appendable, and each time a new column of code is added, its local maxima are sought and connected with existing ridges (or if not found, new ridges are initiated). The ridges fulfilling certain criteria are considered to represent peaks in the spectra.

Ridge detection related functions adapted from: https://github.com/scipy/scipy/blob/v1.5.2/scipy/signal/_peak_finding.py

NOTE: !!! Effect of 'max_seq_len' is not properly tested !!!
