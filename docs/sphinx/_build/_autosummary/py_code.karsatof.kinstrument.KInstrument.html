r_at(x, mode='R', space='mz')¶

Calculate resolution at given m/z or sample number with the 2-parameter resolution function.

Can return resolution (m/dm), peak FWHM or peak sigma in either tof (sample number) or m/z space.

Parameters

  • x (float) -- Peak position as either m/z or TOF sample number, specified by the input parameter 'space'.

  • mode (str, optional) -- Allowed values are 'R', 'fwhm' and 'sigma', by default 'R'.

  • space (str, optional) -- Allowed values are 'mz' and 'tof', by default 'mz'.

Returns

Resolution at 'x'.

Return type

float
