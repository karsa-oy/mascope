py_code.DataVizService¶

Data Visualization Service

This script runs the data visualization service for Karsa Tarkka TOF system.

DataVizService connects to TofDaq Recorder data stream via KAcquisition. It collects MS data and generates images out of it in real time. It also connectes to the MainService via socket, and generates images from data per request.

Running this script will start a KAcquisition instance, and thus requires TofDaq Recorder to be running. At initialization, it will also connect to the MainService. Then the script enters main() function.
