py_code.karsatof.kworker.init_encoders¶

py_code.karsatof.kworker.init_encoders(n_jobs, D, alpha, error_log=False)¶

Initialize KEncoder processes

Parameters

  • n_jobs (int) -- Number of processes to initialize

  • D (csr) -- Peak dictionary sparse matrix

  • alpha (Value) -- SparseCoder regularization parameter

  • error_log (bool, optional) -- Log KEncoder errors to txt files, by default False

Returns

  • list -- List of KEncoder processes

  • Queue -- Queue into which data is put by KFeeder to be processed by KEncoders

  • Queue -- Queue into which KEncoders put the code, to be collected by KCollector

  • list -- List of Event objects, one per KEncoder indicating whether they are currently processing an acquisition.
