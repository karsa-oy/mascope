# HT3000A autosampler agent

### Description

Application for interacting with the HTA 3000A autosampler, and parsing the CSV report files produced by the HTA autosampler manager software.

### NOTE: The application is at a very premature state, and not under active development. It is left here mainly for future reference.

```
mascope
├───agents           # Agent applications
│   ├───file_mover
│   ├───ht300a          # HT3000A autosampler agent
│   │   └───ht3000a         # Source code
│   └───tof_agent
├───backend
├───frontend
└───scripts
```
